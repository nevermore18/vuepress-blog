# 内部查询

### 正在做一个UPDATE/ INSERT/DELETE操作时，不能引用在内部查询表  (可以但是从外部表引用一个字段)

```mysql
UPDATE tablea
SET TIME =
(
    SELECT MAX(TIME)
    FROM tablea # 这是错的
)
WHERE ID = 1;

UPDATE mytable
SET TIME =
(
    SELECT * FROM (
    SELECT MAX(TIME)
    FROM mytable )
    AS tableb    # 这会导致将必需的字段隐式复制到临时表tableb中，因此是允许的
)
WHERE ID = 1;

```
