# 插入排序

```java
public class Insertion {
    private static int[] sort(int[] arr) {
        for (int i = 1; i < arr.length; i++) {
            for (int j = i; j > 0; j--) {
                if (arr[j] < arr[j - 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j - 1];
                    arr[j - 1] = temp;
                }
            }
        }
        return arr;
    }

    public static void main(String[] args) {
        int[] sort = sort(new int[]{2, 1, 3});
        for (int i : sort) {
            System.out.println(i);

        }
    }
}
```
