# 插入排序

```java
public class Insertion {
    
    private static int[] sort(int[] arr) {
        for (int i = 1; i < arr.length; i++) {
            for (int j = i; j > 0; j--) {
                if (arr[j] < arr[j - 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j - 1];
                    arr[j - 1] = temp;
                } else
                    //  如果当前元素比之前的元素大的话直接跳出 因为前面已经是排好序的
                    break;
            }
        }
        return arr;
    }

    public static void main(String[] args) {
        int[] sort = sort(new int[]{4,5,6,1,2,3});
        for (int i : sort) {
            System.out.println(i);
        }
    }
}
```
