# 选择排序

```java
public class Test {

    private static int[] sort(int[] arr) {
        for(int i = 0 ; i < arr.length - 1; i++) {
            int min = i;
            for(int j = i + 1; j < arr.length ; j++) {
                if (arr[j] < arr[min]) {
                    min = j;
                }
            }
            int temp = arr[i];
            arr[i] = arr[min];
            arr[min] = temp;
        }
        return arr;
    }

    public static void main(String[] args) {
        int[] sort = sort(new int[]{2, 1, 3});
        System.out.println(Arrays.toString(sort));
    }
}
```
